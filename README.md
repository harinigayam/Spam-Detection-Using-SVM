# Spam Detection Using Support Vector Machine (SVM)

## 📌 Overview
This project builds a Machine Learning model to classify SMS messages as Spam or Ham (Not Spam) using Support Vector Machines (SVM). 

The system preprocesses text data, converts it into numerical features using TF-IDF, and trains an SVM classifier to accurately detect spam messages.

---

## 🛠 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib / Seaborn
- Google Colab

---

## ⚙️ Project Workflow
1. Data Cleaning & Preprocessing
2. Feature Extraction using TF-IDF Vectorization
3. Train-Test Split
4. Model Training using Support Vector Machine (SVM)
5. Model Evaluation using Accuracy, F1-Score, and ROC-AUC

---

## 📊 Results
- Accuracy: **98%**
- Spam Class F1-Score: **0.94**
- ROC-AUC Score: **0.95**

The model demonstrates strong performance in distinguishing between spam and legitimate messages.
